                                                                                                                                                                      No, right now Fuse will not do a deep search.

                                                                                                                                                                      To get around the issue, you can flatten the array before giving it to Fuse.

                                                                                                                                                                      Example - Convert this:

                                                                                                                                                                      [{foo: 'bar', tags: [{name: 'tag1'}, {name: 'tag2'} }, {foo: 'bzs', tags: [{name: 'foobar'}]}];

                                                                                                                                                                      Into this:

                                                                                                                                                                      [{foo: 'bar', tags: 'tag1 tag2'}, {foo: 'bzs', tags: 'foobar'}];

                                                                                                                                                                      As you can see, the tags have been flattened into a single string value, which can be easily searched by Fuse.
