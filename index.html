Gitsift

Visualizing GitHub public timeline data with D3 and Crossfilter.

Gitsift is a visualization of GitHub public timeline data using D3 and Crossfilter. Most of the code is blatantly copied from Crossfilter's official example. Everybody has to start somewhere. Definition of sift from Merriam-Webster: "to go through (something) very carefully in order to find something useful or valuable". For more information on how to retrieve data from Google BigQuery read our README.
